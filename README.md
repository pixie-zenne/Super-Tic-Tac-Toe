Inspired by Vsauce's Super Tic Tac Toe video.

### The Game
```
   |   |     |   1 | 2 | O   |   O | X | X 
--- --- ---  |  --- --- ---  |  --- --- ---
 X | X |     |   4 | 5 | O   |     | X |   
--- --- ---  |  --- --- ---  |  --- --- ---
   |   |     |   7 | 8 | 9   |     | X |   

-----------     -----------     -----------

 O |   |     |     |   | O   |     | O |   
--- --- ---  |  --- --- ---  |  --- --- ---
   |   | X   |     |   |     |     | X |   
--- --- ---  |  --- --- ---  |  --- --- ---
   |   |     |     | O | O   |     |   |   

-----------     -----------     -----------

   |   |     |     |   |     |     | X | O 
--- --- ---  |  --- --- ---  |  --- --- ---
   |   |     |   O |   |     |     |   |   
--- --- ---  |  --- --- ---  |  --- --- ---
   |   | X   |     |   | X   |   O |   |   


 - | - | X    Current Player : 1 (X)
--- --- ---   Current Mini Board  : 2
 - | - | -    Move 22 : Illegal 
--- --- ---   Select a mini board to mark
 - | - | -    >> 6
```

### Rules
1. To play the game, there must be two players
2. The typical Tic-Tac-Toe rules
